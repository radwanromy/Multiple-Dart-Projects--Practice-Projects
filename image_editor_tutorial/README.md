# Image Editor App

A completely Responsive Image Editor App- Works on Android & iOS! 

## Features
- Pick Image
- Add Text
- Bold Text
- Italic Text
- Add New Line
- Align Text Left, Right & Center
- Change Colour
- Increase Font Size
- Decrease Font Size
